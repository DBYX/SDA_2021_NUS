1. download data from: https://drive.google.com/file/d/1Uy0VmrkbKUAskGKAAQo45F8unrphAF14/view?usp=sharing
and unzip the file, put the data files under 'downloaded_data/data/' folder

2. please install packages in requirements.txt:
pip install -r requirements.txt

3. run python code to get weights (or you can skip since the file is generated for you)
python get_weights.py

4. run python code to run back test and check the asset allocation strategy
python StockCorr_Asset_Allocation.py
